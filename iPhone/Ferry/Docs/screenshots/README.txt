=============================

Upload-ready set: 1284 x 2778 px (matches the "iPhone 6.5" Display" slot in this
app's App Store Connect, which accepts 1242x2688 or 1284x2778). App Store Connect
uses these for all display sizes; only the first 3 appear on install sheets.

Captured on the iOS Simulator (iPhone 17 Pro Max) at 1320x2868, then resized to
1284x2778. Clean 9:41 status bar, light theme. Real app output (the received items
and the price are not mockups).
